Alkom-XML(이미지 발신자표시 전송) 솔루션

이미지발신자표시(Multi-media Caller ID Display) 전송솔루션은 IP 전화기의 디스플레이 창에 전화 발신자의 전화번호가 나타나게 하는 것 뿐만 아니라 전화를 발신한 사람의 사진이미지 및 조직정보 등 다양한 정보를 표시해서 시각화시켜주는 교환기의 부가솔루션입니다.

Alkom-XML 솔루션은 Avaya IP 교환기와 연동하여 IP 전화기의 디스플레이 화면을 통해서 이미지 발신자정보(사진, 조직소속정보 등)를 표시하고, 전화기의 스크린세이버 기능을 통한 본인정보 표시, 회사 조직도와 연계하여 Click-to-Dial/Call 기능, 공지사항 표시 등을 구현할 수 있는 솔루션입니다.
